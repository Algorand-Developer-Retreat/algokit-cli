Usage: algokit init example [OPTIONS] [EXAMPLE_ID]

  Initialize a new project from an example template.

  Allows you to quickly create a new project by copying one of the official
  AlgoKit example templates. If no example ID is provided, launches an
  interactive selector to choose from available examples. The example will be
  copied to a new directory in your current location.

Options:
  -h, --help  Show this message and exit.
